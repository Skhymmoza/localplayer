Документация LocalPlayer API

Добро пожаловать в официальную документацию LocalPlayer API. Здесь вы найдете все необходимое для интеграции VPN функциональности в ваше приложение.

Требования

Для работы с API вам понадобится API ключ. Получите его в API Manager.

Аутентификация

Все запросы к API должны включать ваш API ключ в URL:

Формат запроса 
https://localplayer-tau.vercel.app/api/{YOUR_API_KEY}/endpoint

Base URL

Production 
https://localplayer-tau.vercel.app/api

POST /api/{API_KEY}/connect

Подключить устройство к VPN сети

Параметры запроса

Параметр Тип Обязательный Описание
device_id string Да Уникальный идентификатор устройства
network_uuid string Да UUID сети для подключения
device_name string Нет Человекочитаемое имя устройства

Примеры

Python 
import requests

API_KEY = "LP-a1b2c3d4f5h6"
response = requests.post(
    f"https://localplayer-tau.vercel.app/api/{API_KEY}/connect",
    json={
        "device_id": "android_001",
        "network_uuid": "550e8400-e29b-41d4",
        "device_name": "My Phone"
    }
)
print(response.json())
JavaScript 
const API_KEY = 'LP-a1b2c3d4f5h6';
const response = await fetch(
    `https://localplayer-tau.vercel.app/api/${API_KEY}/connect`,
    {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            device_id: 'android_001',
            network_uuid: '550e8400-e29b-41d4',
            device_name: 'My Phone'
        })
    }
);
const data = await response.json();

POST /api/{API_KEY}/disconnect

Отключить устройство от VPN сети

Python 
response = requests.post(
    f"https://localplayer-tau.vercel.app/api/{API_KEY}/disconnect",
    json={"device_id": "android_001"}
)
print(response.json())

GET /api/{API_KEY}/devices

Получить список всех подключенных устройств в сети

Python 
response = requests.get(
    f"https://localplayer-tau.vercel.app/api/{API_KEY}/devices"
)
devices = response.json()
for device in devices['devices']:
    print(f"{device['name']}: {device['ip']}")

Ответ

Проверить статус сети и API ключа

Запрос

Коды ошибок

Код Описание Решение
400 Bad Request Проверьте параметры запроса
401 Unauthorized Неверный API ключ
429 Too Many Requests Превышен лимит запросов
500 Server Error Повторите запрос позже
