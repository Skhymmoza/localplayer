Часто задаваемые вопросы

Как получить API ключ?

Перейдите в раздел API Manager и создайте новый ключ, указав UUID вашей сети.

Какие есть лимиты на бесплатном плане?

Бесплатный план включает 100 запросов в час и поддержку до 10 устройств.

Поддерживается ли iOS?

В данный момент LocalPlayer API оптимизирован для Android. Поддержка iOS запланирована на 2026 год.

Как работает биллинг?

Оплата производится ежемесячно. Вы можете обновить или отменить подписку в любое время.

Есть ли SLA гарантии?

Да, для Enterprise тарифа мы гарантируем SLA 99.99% uptime.

Могу ли я использовать API в коммерческих проектах?

Да, вы можете использовать LocalPlayer API в любых проектах, включая коммерческие.
